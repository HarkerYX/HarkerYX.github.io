有时需要在linux服务器与windows系统间传递文件，samba服务是可以实现的，但是有点繁琐，这里介绍一个工具，安装rz sz命令后可以很方便的实现文件互传，在命令行输入命令就可以拖拽上传下载，非常实用方便！

- **安装：**

```elm
sudo apt-get install lrzsz
```

- **rz : 下载文件**
